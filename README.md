# `@uanbr/prettier-config`

> My personal [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
yarn add --dev @uanbr/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@uanbr/prettier-config"
}
```

## Publicando do Yarn

```bash
yarn login
yarn publish --access public
```
